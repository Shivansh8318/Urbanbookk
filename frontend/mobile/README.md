## Installation Guide

### 1. Clone the repository

```sh
git clone https://github.com/Shivansh8318/UrbanBookFrontendd.git
cd UrbanBook6/UrbanBookk
```

### 2. Install dependencies

```sh
# Using npm
npm install

# OR using Yarn
yarn install
```

npx react-native run-android
